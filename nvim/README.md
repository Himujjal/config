## Neovim Philosophy

Neovim is a modern fork of Vim that preserves its core modal editing paradigm while embracing extensibility and performance. Its philosophy centers on:

- **Extensibility via Lua** – Lua is a first‑class scripting language, making plugins faster and easier to write than with Vimscript.
- **Asynchronous by design** – Built‑in job control and RPC enable non‑blocking operations like LSP, linting, and file watching.
- **User freedom** – No enforced defaults; you configure Neovim exactly how you want, with batteries included but not mandatory.
- **Community‑driven** – Rapid iteration, open development, and a strong focus on backward compatibility when possible.
- **Performance** – Cleaner C codebase, reduced memory footprint, and faster startup compared to legacy Vim.

Ultimately, Neovim aims to be the editor you grow with: simple to start, infinitely customizable as your needs evolve.
