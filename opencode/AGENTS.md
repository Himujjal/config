# Opencode repo

Refer to [opencode-config-reference.md](./docs/opencode-config-reference.md) for config reference and other stuffs.

## Search

Use `rg` (ripgrep) instead of the built-in grep tool for content searches. Run `rg <pattern>` via bash rather than calling the grep tool.

## Zig

When the user asks for zig help or documentation, use `zigdoc -h <topic>` instead of any built-in help mechanisms.
